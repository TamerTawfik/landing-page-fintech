<!-- GETTING STARTED -->
## Getting Started

Make sure Node.js is installed in your OS.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TamerTawfik/landing-page-fintech.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. start the local server
   ```sh
   npm run start
   ```
